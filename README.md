# Visibilidade e Transparência para as votações dos Projetos de Lei

O projeto terá como objetivo inicial desenvolver um conjunto de funcionalidades essenciais que permitam coletar, organizar e disponibilizar informações públicas de forma transparente e acessível.

*Coleta automatizada de dados públicos:* 

desenvolvimento de crawlers e scripts responsáveis por obter informações de fontes públicas oficiais.

*Organização e estruturação dos dados:*

criação de um sistema para armazenar e organizar os dados coletados de maneira clara e pesquisável.

*Garantia de integridade e confiabilidade das informações:*

implementação de mecanismos de auditoria e/ou assinaturas digitais para assegurar que os dados apresentados não foram alterados, evitando que o usuário precise apenas confiar na plataforma.

*Plataforma acessível e intuitiva:*

desenvolvimento de uma interface simples e fácil de navegar, permitindo que qualquer usuário identifique rapidamente temas de interesse, como votações de projetos de lei e os políticos responsáveis ou envolvidos nesses projetos.
