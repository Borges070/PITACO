# PITACO
Pitaco é um espaço de aprendizado vivo, horizontal e colaborativo, onde todas as pessoas têm voz ativa e participam de forma conjunta no processo de construção do conhecimento. Cada ciclo representa uma nova oportunidade de colocar a mão na massa, experimentar na prática, aprender em conjunto e transformar ideias em projetos reais e concretos. Esta oficina funciona como um ambiente constante de troca, diálogo e colaboração, onde o conhecimento não é imposto, mas sim construído coletivamente a partir das experiências, vivências e contribuições de cada participante.
